# Card RPG 시스템 구현 매뉴얼 (코드 동기화판)

이 문서는 `card/` 버전의 실제 동작 코드(`index.html`, `logic.js`, `data.js`)와 **1:1로 맞춘 구현 명세**입니다.  
아래 규칙만 따르면 동일한 시스템을 재구현할 수 있습니다.

---

## 0. 코드 동기화 정오표 (필독)

- **MP 자동 회복(턴 시작 +20 / 피격 +10 / 행동 +10)은 현재 실제 전투 루프에 적용되지 않습니다.** 해당 상수는 정의되어 있으나 실제 턴 처리에서 호출되지 않습니다.
- **달의축복(`moon_bless`) 자체에는 관통이 없습니다.** 기본 효과는 `MATK +30%`, `회피율 +15%`입니다.
- 관통은 개별 스킬/특성/아티팩트에서만 발생합니다(예: `flame_piercing`, `divine_piercing`, 일부 초월 스킬 처리).
- 대미지 방어식의 방어 상수는 `100`이며, 최종식은 `finalDmg = floor(val * finalMult * (100 / (100 + def)))` 입니다.
- 카오스 룰렛(초월 카드), 전투 후 퀴즈 연동 보상, 아티팩트 선택 분기는 모두 실제 구현되어 있으므로 구현 명세에 반드시 포함해야 합니다.

## 1. 전투 시스템 및 스탯 구조

### 1-1. 스탯 (Stats)
모든 아군 캐릭터와 적군은 다음과 같은 스탯을 보유합니다.
- **HP (체력)**: 소진 시 사망.
- **MP (마나)**: 스킬 사용에 소모되는 재화. 최대치 100. (**현재 실동작은 자동 회복 없음**: 턴 시작/피격/행동 시 자동 +MP 로직 미적용)
- **ATK (물리공격력) & MATK (마법공격력)**: 각각 물리 스킬, 마법 스킬의 기반 대미지.
- **DEF (물리방어력) & MDEF (마법방어력)**: 적의 공격을 감소시키는 수치.
- **Crit (치명타 확률)**: 기본치 10%. 발동 시 최종 대미지가 1.5배 상승.
- **Evasion (회피율)**: 기본치 5%. 발동 시 적의 공격을 완전히 무효화함. (100분율로 계산)

## 1) 데이터 구조

## 1-1. 카드 원본 데이터
- 카드 정의는 `data.js`의 `CARDS`, `BONUS_CARDS`, `TRANSCENDENCE_CARDS`.
- 각 카드 스키마:
  - `id, name, grade, element, role`
  - `stats: { hp, atk, matk, def, mdef }`
  - `trait: { type, val?, ... }`
  - `skills: [{ name, type(phy/mag/sup), tier, cost, val?, desc, effects[] }]`

## 1-2. 전투 중 캐릭터 인스턴스
- 전투 복사본은 대략 아래 필드 사용:
  - `hp, maxHp, mp, atk, matk, def, mdef`
  - `baseCrit(기본 10), baseEva(기본 0)`
  - `buffs` (맵), `isDead`, `proto`(원본 카드 참조)

## 1-3. 필드 버프
- 기본 동시 유지 상한: **3개**, 아티팩트 `buff_overload` 보유 시 **5개**.
- 정의값(플레이어 측에만 스탯 반영):
  - `sun_bless`: atk+30%, matk+30%
  - `moon_bless`: matk+30%, evasion+15
  - `sanctuary`: matk+30%, mdef+30%
  - `goddess_descent`: atk/matk/def/mdef +30%
  - `earth_bless`: atk+25%, matk+25%
  - `twinkle_party`: atk+20%, crit+15
  - `star_powder`: def+40%, mdef+40%
  - `reaper_realm`: crit+40
  - `gale`: crit+20, evasion+20

## 2) 스탯 계산 규칙 (`Logic.calculateStats`)

최종 스탯 계산 순서:

1. **기본값 로드**  
   - `crit = baseCrit(없으면 10)`  
   - `evasion = baseEva + 5`

2. **혼돈 축복(blessing) 보정**  
   - 축복 받은 카드: crit +10, evasion +5 (추가로 전투 시작 전 스탯 배수도 이미 반영됨)

3. **조건형 특성(합연산)**  
   - 예: 필드버프 없음 시 crit/evasion 증가 등.

4. **배율 묶음 m 초기값 1.0**  
   - `m = { atk:1, matk:1, def:1, mdef:1 }`
   - 필드버프/디버프/아티팩트/일부 특성은 전부 여기에 더하거나 빼서 누적.

5. **필드버프 적용(플레이어만)**  
   - 각 버프 수치를 `m.*` 또는 `crit/evasion`에 누적.
   - 모드 `flood`이면 필드버프 수치 2배.
   - 아티팩트 배율:
     - `nature_blessing`: `earth_bless` 효과 2배
     - `milkshake`: `star_powder` 효과 2배

6. **개인 버프/디버프 적용**
   - `weak`: atk -20%
   - `silence`: matk -20%
   - `curse`: mdef -20%
   - `evasion`: evasion +50
   - `darkness/corrosion`: def -20% / 동시 -40%
   - 모드 `curse`이면 위 디버프 계수 2배

7. **관련 아티팩트 보정**
   - `assassin_nail`: darkness/corrosion 방어 감소량 2배
   - `shadow_ball`: darkness가 mdef도 감소
   - `veil_of_darkness`: 어둠 속성 crit/evasion +10
   - `rabbit_hole`: 특정 토끼 3종 crit/evasion +20
   - `shadow_stab`: evasion +20, def/mdef -30%

8. **최종치 확정**
   - `finalStat = floor(base * max(0, m))`

### 1-5. 구현 고정 계산식 (정확)
- **스탯 계산**: 기본 스탯에 버프/디버프/특성/아티팩트를 순차 반영한 뒤 `floor` 처리.
- **치명타 배율**: 기본 1.5, `sun_bless` 시 +0.6, `reaper_realm` 시 +0.4.
- **속성 상성**: 우위 시 1.2배.
- **최종 대미지**:
  - `finalMult = skillMult * (1 + dmgBonus)`
  - `finalDmg = floor(val * finalMult * (100 / (100 + def)))`
- **달의축복 자체 관통 없음**: 관통은 스킬/특성/아티팩트 트리거에서 개별 적용.

## 2. 상태이상 및 필드버프

전투의 핵심은 적과 아군에게 걸리는 개인 상태이상(버프/디버프)과 전장을 덮는 필드버프의 계산입니다.

### 2-1. 전투 필드버프 (Field Buffs)
필드에는 동시에 최대 3개(아티팩트에 따라 최대 5개)의 필드버프가 덮일 수 있으며, 아군에게만 중첩 상승 효과를 줍니다.
- **태양의축복**: ATK / MATK 30% 증가. 치명타 배율이 기존 1.5배에서 2.1배(+0.6)로 고정 상승. (추가로 초월 루미의 스킬 등 특정 스킬 발동 시 확정 치명타 및 위력 증가 트리거)
- **달의축복**: MATK 30%, 회피율 15% 증가. (**버프 자체에는 관통 없음**. 관통은 별도 스킬/아티팩트 효과에서만 발생)
- **성역**: MATK / MDEF 30% 증가.
- **여신강림**: 물리와 마법의 공/방 모두(ATK, MATK, DEF, MDEF) 30% 증가.
- **대지의축복**: ATK / MATK 25% 증가.
- **트윙클파티**: ATK 20%, 치명타율 15% 증가.
- **스타파우더**: DEF / MDEF 40% 대폭 증가.
- **사신강림**: 치명타율 40% 증가. 치명타 배율 +0.4 (1.9배). (추가로 초월 형태 스킬 발동 시 50%의 마방 관통 효과를 부르는 트리거로 작용)
- **질풍**: 치명타율 20%, 회피율 20% 증가.

*(주의: 여러 개가 활성화되면 각각의 효과가 합산되어 계산됩니다.)*

### 2-2. 개인 버프 & 디버프
- **암흑 & 부식 (Darkness, Corrosion)**: 각각 방어력을 20% 깎습니다. 두 개가 동시에 걸리면 40%가 깎입니다.
- **약화 (Weak)**: 물리 공격력(ATK) 20% 감소.
- **침묵 (Silence)**: 마법 공격력(MATK) 20% 감소.
- **저주 (Curse)**: 마법 방어력(MDEF) 20% 감소.
- **작열 (Burn) & 디바인 (Divine)**: 최대 3스택(아티팩트 시 5스택)이 쌓이는 특수 재화입니다. 특정 스킬이 이 스택을 소모하여 강력한 대미지 배율을 만듭니다.
- **기절 (Stun)**: 부여된 턴에 어떠한 행동도 할 수 없습니다.
- **가드 / 매직가드 / 배리어**: 각각 받는 모든 대미지 반감(50%) / 마법 피해 완전 무효 / 물리 피해 완전 무효.
- **회피태세**: 발동 즉시 회피율 50%를 고정 상승.

## 4) 전투 턴 루프 요약

1. 플레이어 턴 시작
   - 기절(`stun`)이면 행동 불가 후 해제.
   - 지연 스킬 발동 체크.
   - 아티팩트/필드 관련 턴 시작 효과 처리(`kaleidoscope`, `gale_storm` 등).

2. 플레이어 행동
   - 스킬 선택(또는 일반공격).
   - MP 검사 후 사용(예외: `blue_moon` 30% 확률 무소모, `support_boost`로 sup cost 0).
   - 명중 전 회피 판정.
   - 배리어/매직가드/가드 반영.
   - 대미지 및 부가효과 적용.

3. 적 턴
   - 보스 전용 AI/방어 프로토콜 처리.
   - 동일하게 회피/보호/대미지 계산.

4. 사망 처리
   - 사망 특성(`handleDeathTraits`) 발동.
   - 다음 슬롯 교대.

5. 종료
   - 적 전멸 시 승리, 아군 전멸 시 패배.

## 5) MP(마나) 시스템 실제 동작

- 시작 MP: 100
- 스킬 사용 시 `cost`만큼 차감
- 예외:
  - `blue_moon`: 30% 확률로 MP 미소모
  - `support_boost`: 보조스킬 `sup` 비용 0
  - `lucky_vicky`: 치명타 또는 회피 시 MP +10
  - 일부 효과로 즉시 MP 증가(예: 꿈의형태에서 성역/스타파우더 변환 시)
- **자동 MP 회복 루프(턴/피격/공격)는 현재 없음**

### 3-4. 퀴즈 연동 보상 시스템 (실구현)
- 일반 승리 후 추가 퀴즈 성공 시 드로우권 +1.
- `creator_god` 격파 시:
  - 일반 모드: 문법 퀴즈 성공 시 드로우권 +3.
  - artifact 모드: 문법 퀴즈 성공 시 아티팩트 선택 기회 획득.
- chaos/draft 모드: 콜로케이션 퀴즈 성공 시 드로우권 +1.

## 6) 모드 / 가챠 / 초월 / 퀴즈 연동

## 6-1. 시작 티켓
- 기본 20, 대부분 특수 모드 10, draft 5, chaos 0.

## 6-2. 가챠 등급 확률
- 모드별 확률표 `GACHA_RATES` 사용.
- challenge 뽑기는 일반보다 상위 등급 확률 증가.

## 6-3. 카오스 룰렛(초월 카드 획득)
- `chaosTickets` 1장 소모.
- 아직 `pendingTranscendenceCards`에 없는 초월 카드 중 랜덤 1장 지급.

## 6-4. 혼돈의 축복 / 대현자의 축복
- 혼돈 축복: 전투 내 강화 버프 부여, 사용 횟수 제한.
- 대현자 축복: 선택 카드군(최대 12) 강화 + 보상(티켓) 로직 포함.

## 6-5. 퀴즈 연동 보상
- 일반 승리 후: 퀴즈 성공 시 티켓 +1.
- `creator_god` 처치 시:
  - 일반 모드: 문법 퀴즈 성공 시 티켓 +3
  - artifact 모드: 문법 퀴즈 성공 시 아티팩트 선택 기회
- chaos/draft: 콜로케이션 퀴즈 성공 시 티켓 +1

> 따라서 “퀴즈 연동 강화/보상 시스템”은 **명세에 반드시 포함**해야 합니다.

## 5. 아티팩트 사전 (Artifact Objects)

아티팩트 모드에서 얻을 수 있는 특수 장비와 명확한 효과들입니다. (**현재 구현 기준 총 26개**)

- **대자연의 축복 (`nature_blessing`)**: 필드버프 `대지의축복` 효과 2배.
- **리버스 (`reverse`)**: 자연속성 카드 사망 시 `대지의축복` 부여.
- **밀크쉐이크 (`milkshake`)**: `스타파우더` 효과 2배.
- **버프오버로드 (`buff_overload`)**: 필드버프 상한 5개.
- **섀도우볼 (`shadow_ball`)**: `암흑`이 마법방어력도 감소.
- **어쌔신네일 (`assassin_nail`)**: 암흑/부식 방어 감소 효과 2배.
- **베일오브다크니스 (`veil_of_darkness`)**: 어둠속성 카드 치명타/회피 +10%.
- **래빗홀 (`rabbit_hole`)**: 눈토끼/밤토끼/은토끼 치명타/회피 +20%.
- **럭키비키 (`lucky_vicky`)**: 치명타 또는 회피 발생 시 MP +10.
- **오버플레임 (`over_flame`)**: 작열 최대 5스택, 부여량 2스택.
- **오버디바인 (`over_divine`)**: 디바인 최대 5스택, 부여량 2스택.
- **홀리플레임버스트 (`holy_flame_burst`)**: 작열/디바인 전소모형 스킬 추가위력 2배.
- **플레임피어싱 (`flame_piercing`)**: 작열 스택당 물리방어 10% 관통.
- **디바인피어싱 (`divine_piercing`)**: 디바인 스택당 마법방어 10% 관통.
- **질풍노도 (`gale_storm`)**: 전투 시작 후 3턴간 `질풍` 부여.
- **프로즌바디 (`frozen_body`)**: 물속성 카드 사망 시 적 스턴.
- **아이스브레이크 (`ice_break`)**: 스턴 상태 적에게 대미지 3배.
- **서포트부스트 (`support_boost`)**: 모든 보조스킬 마나 비용 0.
- **더블어택 (`double_attack`)**: 일반공격 위력 2배.
- **데스룰렛 (`death_roulette`)**: 모든 스킬 대미지 2배, 스킬 사용 시 30% 확률 즉사.
- **섀도우스탭 (`shadow_stab`)**: 회피 +20%, DEF/MDEF -30%.
- **드래곤하트 (`dragon_heart`)**: 베이비/레드/골드드래곤 마공 +50%.
- **빅뱅 (`big_bang`)**: 전설/초월 카드 사망 시 물리 3배율 자폭대미지.
- **길동무 (`companion`)**: 사망 유발 특성/아티팩트 대미지 2배.
- **만화경 (`kaleidoscope`)**: 매 턴 시작 시 모든 필드버프 변경.
- **블루문 (`blue_moon`)**: 스킬 사용 시 30% 확률로 마나 미소모.
