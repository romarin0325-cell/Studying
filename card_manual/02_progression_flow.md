# 2편 진행 방식

## 1. 새 런 시작

새 게임을 시작하면 아래 요소가 함께 초기화된다.

- 현재 티켓 수
- 인벤토리와 3칸 덱
- 현재 스테이지와 다음 적
- 이번 런에서만 쓰는 축복, 아티팩트, 초월 후보, 이벤트 카드
- 카오스/드래프트 전용 임시 카드 풀

진행 형태는 두 가지다.

- 일반 진행: 정해진 클리어 스테이지까지 돌파
- 엔드리스 진행: 계속해서 다음 보스를 상대하며 해금과 기록을 노림

## 2. 초보자 보정

- 해금 보너스 카드가 5장 미만인 초반 계정은 안전장치가 적용된다.
- 일반 진행에서는 루미가 기본 인벤토리에 추가될 수 있다.
- 카오스와 드래프트는 시작 티켓이 너무 적지 않도록 보정된다.

## 3. 전투 전 준비

### 가챠

- 일반 가챠는 티켓 1장을 사용한다.
- 챌린지 가챠는 티켓 1장을 먼저 쓰고 퀴즈 성공 시 더 좋은 확률표를 적용한다.
- 모드별 등급 확률은 9편 표를 기준으로 읽으면 된다.

### 덱 편집

- 덱은 반드시 3칸을 채워야 전투에 들어갈 수 있다.
- 보너스 카드 편집은 "이번 런에서 어떤 보너스 카드를 등장시킬지"를 정하는 메타 설정이다.

### 축복과 초월

- 혼돈의 축복은 일반 3장, 도전 5장 후보 중 하나를 고른다.
- 대현자의 축복은 문법 퀴즈 성공 시 최대 12장 후보 중 하나를 고른다.
- 카오스는 이번 런 전용 카드 15장 풀을 만들어 계속 갈아끼운다.
- 드래프트는 4장 선택지를 보고 리롤을 사용해 최종 카드를 고른다.

## 4. 적 등장 흐름

- 기본 보스 순환은 인조 마신 → 사랑의 여신 아이리스 → 저주의 여신 아이리스 → 고대신 파라오 → 마신 벨제뷔트 → 창조신 아스테아다.
- 엔드리스 37스테이지 이후에는 대응 숨은 보스가 30% 확률로 끼어든다.
- 시즌 스페셜 미션이 열려 있으면 엔드리스에서 시즌 보스가 끼어들 수 있다.

## 5. 승리 후 처리

공통 보상:

- 티켓 지급
- 월간/주간/시즌 미션 진행도 반영
- 엔드리스 후반 해금 판정 반영

보스 승리 시 추가로 생길 수 있는 일:

- 창조신 문법 퀴즈
- 아티팩트 모드 아티팩트 선택
- 숨은 보스 보너스 초월 해금
- 엔드리스 37스테이지 이후 신기 해금
- 시즌 미션 보스 카운트 증가

모드별 추가 처리:

- 카오스: 다음 전투용 카드 15장을 다시 구성
- 드래프트: 드래프트 상태 초기화 후 다음 선택지 준비
- 꿈의회랑: 전용 종료 루틴으로 넘어가며 숨은 학습 흐름을 갱신

## 6. 패배 후 처리

- 일반 진행은 런 정리 후 메뉴로 돌아간다.
- origin, chaos, draft, 꿈의회랑은 각자 전용 정리 규칙이 있어 예약 초월 카드나 숨은 학습 카운트가 함께 초기화될 수 있다.
- 글로벌 해금, 티켓, 미션, 신기 해금은 별도 저장 데이터에 남는다.

## 7. 실전마법연습과 꿈의회랑

- 실전마법연습은 월간/주간 미션을 함께 올린다.
- 숨은 학습 카운트가 특정 조건에서만 올라가며, 5회 누적 시 꿈의회랑 진입 준비가 끝난다.
- 꿈의회랑은 시즌 스페셜 미션과 메타 해금의 중요한 분기점이다.
