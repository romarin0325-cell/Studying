# 5편 특수카드데이터

## 범위

이 문서는 `BONUS_CARDS`, `TRANSCENDENCE_CARDS`, `BONUS_TRANSCENDENCE_CARDS`를 다룬다.

현재 버전의 핵심 보정점:

- 보너스 카드가 33장 -> 35장으로 증가
- 신규 보너스 카드: `phantom`, `sylphid`
- 기본 초월 카드가 8장 -> 7장으로 재편
- `trans_gray`가 보너스 초월 쪽으로 이동
- 신규 보너스 초월: `trans_flora`

스탯 표기는 `HP/ATK/MATK/DEF/MDEF` 순서다.
