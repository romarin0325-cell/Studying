# 11편 대미지 계산 상세 공식

이 문서는 실제 코드 순서를 기준으로 대미지를 역산할 수 있도록 정리한 상세판입니다.

## 1. 전투용 기본 스탯이 만들어지는 순서

전투 시작 전에 카드 기본 스탯에 아래 순서로 보정이 들어간다.

1. 카드 원본 스탯 복사
2. 덱 조건 특성
3. 배치 특성
4. 토끼/슬라임 같은 개체 수 특성
5. 파티 전체 특성
6. 드래곤하트 같은 런 전체 아티팩트
7. 각 단계마다 소수점은 내림

기본 치명타율은 10%, 기본 회피율은 5%다.

## 2. 실시간 스탯 계산식

공격 직전에 다시 계산하는 값은 아래와 같다.

### 공격/방어 계열

```text
최종 스탯 = floor(기본 스탯 × max(0, 1 + 필드버프 합 + 조건부 특성 합 - 디버프 합 + 아티팩트 합))
```

핵심 규칙:

- 같은 스탯에 붙는 필드버프는 합연산이다.
- 같은 스탯에 붙는 디버프도 합연산이다.
- 플러드 모드는 필드버프 수치를 2배로 만든다.
- 커스 모드는 디버프 감소량을 2배로 만든다.
- 암흑 + 부식은 DEF -40%로 합쳐진다.
- 어쌔신네일이 있으면 이 감소량이 2배가 되어 최대 DEF -80%가 된다.

### 치명타율 / 회피율

```text
치명타율 = 10 + 축복 보너스 + 필드버프 보너스 + 특성 보너스 + 아티팩트 보너스
회피율 = 5 + 축복 보너스 + 필드버프 보너스 + 특성 보너스 + 1턴 버프 + 아티팩트 보너스 - 신기 파라오
```

추가 규칙:

- 축복 보너스: 치명타율 +10, 회피율 +5
- 회피태세: 회피율 +50
- 질풍: 치명타율 +20, 회피율 +20
- 달의축복: 회피율 +15
- 신기 파라오를 가진 런은 마지막에 회피율 50을 빼고, 결과가 0 아래면 0으로 고정한다.

## 3. 상성 배율

상성 우위만 존재하며 불리 상성 감산은 없다.


### 물
- **우위 대상**: 불
- **배율**: 1.2

### 불
- **우위 대상**: 자연
- **배율**: 1.2

### 자연
- **우위 대상**: 물
- **배율**: 1.2

### 빛
- **우위 대상**: 어둠
- **배율**: 1.2

### 어둠
- **우위 대상**: 빛
- **배율**: 1.2

카오스로드의 "모든 적에게 상성 우위" 특성은 언제나 1.2배로 처리한다.

## 4. 플레이어 공격 대미지 계산 순서

### 4-1. 치명타 판정

- 먼저 치명타 여부를 굴린다.
- 확정 치명타, 추가 치명타 확률 효과가 있으면 이 단계에서 덮어쓴다.
- 치명타 배율은 기본 1.5배다.
- 태양의축복이 있으면 +0.6, 사신강림이 있으면 +0.4가 더해진다.

### 4-2. 공격력 원본 결정

```text
물리 스킬이면 ATK 사용
마법 스킬이면 MATK 사용
상성 우위면 여기서 1.2배
치명타면 마지막에 치명타 배율 곱셈
```

### 4-3. 스킬 배율 누적

```text
mult = 스킬의 기본 배율
```

여기에 아래가 더해지거나 곱해진다.

- "스택당 +n배"는 `mult`에 더한다.
- "조건 충족 시 2배" 같은 문구는 `mult *= 2.0`처럼 곱한다.
- 일반 공격 + 아레나는 `mult *= 2.0`
- 일반 공격 강화 특성도 `mult *= 강화배율`
- 데스룰렛은 이 단계 끝에서 `mult *= 2.0`

### 4-4. 특성 추가 피해

일부 특성은 `dmgBonus`라는 별도 항으로 쌓인다.

```text
finalMult = mult × (1 + dmgBonus)
```

즉:

- 스킬 자체 배율과 특성 추가 피해는 곱연산이다.
- 예를 들어 기본 2.5배 스킬 + 특성 추가 피해 50%면 `2.5 × 1.5 = 3.75배`가 된다.

## 5. 방어력, 디버프, 관통 처리 순서

### 5-1. 실제 적용 DEF/MDEF

```text
def = 실시간 계산이 끝난 최종 DEF 또는 MDEF
```

이 값에는 필드버프, 디버프, 특성, 아티팩트가 이미 반영되어 있다.

### 5-2. 관통 기준 방어 상수

관통이나 "방어력 n% 무시"는 별도의 기준값을 쓴다.

```text
rawDef = 대상의 현재 DEF
rawMdef = 대상의 현재 MDEF
관통량 = floor(rawDef 또는 rawMdef × 관통률)
```

즉:

- 최종 `def`는 실시간 계산값을 사용한다.
- 관통량 계산용 상수는 대상에게 현재 걸려 있는 기본 DEF/MDEF 값을 사용한다.
- 적 턴 시작에 보스 패턴으로 DEF/MDEF가 바뀌면 그 바뀐 값이 `rawDef`, `rawMdef`가 된다.

### 5-3. 관통은 이렇게 누적된다

```text
def = 최종 DEF/MDEF
def -= 각종 관통량
def = max(0, def)
```

따라서 방어 디버프와 관통은 서로 다른 단계에서 같이 작용한다.

- 방어 디버프: 실시간 DEF/MDEF 자체를 낮춘다.
- 관통: 낮아진 DEF/MDEF에서 추가로 고정량을 뺀다.

## 6. 최종 대미지 공식

```text
최종 대미지 = floor(공격력 원본 × finalMult × (100 / (100 + def)))
```

여기서:

- 공격력 원본 = ATK 또는 MATK, 상성 보정 포함, 치명타면 치명타 배율 포함
- `finalMult = mult × (1 + dmgBonus)`
- `def` = 실시간 계산 후 관통까지 뺀 DEF/MDEF

## 7. 스킬 후처리 타이밍

공격 스킬은 대부분 아래 순서로 처리된다.

1. 대미지 계산
2. 대미지 적용
3. 소모형 후처리 적용
4. 스킬에 적힌 버프/디버프/필드버프 부여
5. 피격 반응 특성 처리

중요한 의미:

- "작열을 전부 소모하고 대미지 증가"는 이번 공격에 즉시 반영된다.
- "적에게 작열 부여"는 이번 공격 계산에는 보통 포함되지 않고, 공격 후에 붙는다.
- 필드버프 제거형 스킬도 계산 시 가상으로 먼저 제거를 반영하고 실제 제거는 피해 직후 적용한다.

## 8. 적이 플레이어를 때릴 때의 순서

적 공격은 플레이어 공격과 순서가 조금 다르다.

1. 대상 회피 판정
2. 배리어 / 매직가드 판정
3. 가드 판정
4. 대미지 계산 및 적용
5. 스킬 부가효과 적용

즉 배리어, 매직가드, 가드는 "적 턴에 플레이어가 방어할 때" 정확히 작동한다.

## 9. 예시 계산

예시: 불속성 물리 카드가 자연속성 적에게 기본 2.5배 스킬을 사용하고, 작열 3스택 대상에게 신기 아레스 없이 플레임피어싱이 있는 경우

1. 상성 우위로 공격력 원본에 1.2배
2. 치명타가 아니면 그대로 진행
3. `mult = 2.5`
4. 실시간 DEF 계산 후, 플레임피어싱으로 `rawDef × 30%`만큼 추가 차감
5. 최종 공식 `floor(ATK × 1.2 × 2.5 × 100/(100+def))`

예시: 마법 카드가 디바인 4스택 대상에게 신기 아이리스를 들고 공격하면

1. 디바인 4스택으로 현재 공격 타입 기준 방어를 40% 관통
2. 마법이면 `rawMdef × 40%`를 깎는다
3. 그 뒤 `floor(MATK × finalMult × 100/(100+def))`
