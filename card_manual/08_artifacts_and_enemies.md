# 8편 아티팩트·보스·적 패턴

## 1. 아티팩트 개요

- 기본 아티팩트는 26종이다.
- 아티팩트 모드에서는 한 런에 최대 4개까지 획득한다.
- 신기 10종이 별도로 정의되어 있다.
- 실제 아티팩트 모드 선택 풀은:
  - 기본 26종에서 해금된 신기가 기존 아티팩트를 대체한다.
  - 신기 파라오는 새로 추가된다.
  - 그래서 최대 풀 크기는 27종이다.

## 2. 기본 아티팩트


### 대자연의 축복
- **효과**: 대지의축복 효과 2배

### 리버스
- **효과**: 자연속성 카드 사망 시 대지의축복 발동

### 밀크쉐이크
- **효과**: 스타파우더 효과 2배

### 버프오버로드
- **효과**: 필드버프 최대 5개

### 섀도우볼
- **효과**: 암흑이 MDEF도 감소

### 어쌔신네일
- **효과**: 암흑/부식에 의한 방어 감소 2배

### 베일오브다크니스
- **효과**: 어둠속성 카드 치명타율 +10%, 회피율 +10%

### 래빗홀
- **효과**: 눈토끼, 밤토끼, 은토끼 치명타율 +20%, 회피율 +20%

### 럭키비키
- **효과**: 치명타 또는 회피 성공 시 MP 10 회복

### 오버플레임
- **효과**: 작열 최대 5스택, 부여량 2스택

### 오버디바인
- **효과**: 디바인 최대 5스택, 부여량 2스택

### 홀리플레임버스트
- **효과**: 작열/디바인 전소모 스킬 추가위력 2배

### 플레임피어싱
- **효과**: 작열 스택당 물리 DEF 10% 관통

### 디바인피어싱
- **효과**: 디바인 스택당 마법 MDEF 10% 관통

### 질풍노도
- **효과**: 전투 시작 시 3턴 동안 질풍

### 프로즌바디
- **효과**: 물속성 카드 사망 시 적 기절

### 아이스브레이크
- **효과**: 기절 대상 대미지 2배

### 서포트부스트
- **효과**: 보조 스킬 MP 소모 0

### 더블어택
- **효과**: 일반 공격 대미지 2배

### 데스룰렛
- **효과**: 모든 스킬 대미지 2배, 스킬 사용 시 30% 확률 사망

### 섀도우스탭
- **효과**: 회피율 +20%, DEF/MDEF -30%

### 드래곤하트
- **효과**: 베이비드래곤, 레드드래곤, 골드드래곤, 에인션트드래곤 MATK 2배

### 빅뱅
- **효과**: 전설/초월 카드 사망 시 물리 3배율 자폭

### 길동무
- **효과**: 사망 반격/자폭 계열 대미지 2배

### 만화경
- **효과**: 턴 시작마다 필드버프 재구성

### 블루문
- **효과**: 스킬 사용 시 30% 확률로 MP 미소모

## 3. 신기 해금표

엔드리스 모드에서 37스테이지 이후 보스를 잡으면 아래 신기가 영구 해금될 수 있다.


### 사랑의 여신 아이리스
- **신기**: 신기 아이리스
- **해금 확률**: 1%
- **아티팩트 모드 처리**: 디바인피어싱 대체
- **효과**: 디바인 스택당 DEF/MDEF 10% 추가 관통

### 저주의 여신 아이리스
- **신기**: 마신기 아이리스
- **해금 확률**: 1%
- **아티팩트 모드 처리**: 플레임피어싱 대체
- **효과**: 작열 스택당 DEF/MDEF 10% 추가 관통

### 고대신 파라오
- **신기**: 신기 파라오
- **해금 확률**: 1%
- **아티팩트 모드 처리**: 신규 추가
- **효과**: DEF +50%, MDEF +50%, 회피율 -50%(최저 0%)

### 마신 벨제뷔트
- **신기**: 마신기 벨제뷔트
- **해금 확률**: 1%
- **아티팩트 모드 처리**: 빅뱅 대체
- **효과**: 전설/초월 카드 사망 시 물리 4배율 자폭

### 뇌신 토르
- **신기**: 신기 토르
- **해금 확률**: 1%
- **아티팩트 모드 처리**: 밀크쉐이크 대체
- **효과**: 스타파우더 효과 2.5배

### 꽃의 여신 플로라
- **신기**: 신기 플로라
- **해금 확률**: 1%
- **아티팩트 모드 처리**: 대자연의 축복 대체
- **효과**: 대지의축복 효과 2.5배

### 사신 그레이
- **신기**: 신기 그레이
- **해금 확률**: 1%
- **아티팩트 모드 처리**: 베일오브다크니스 대체
- **효과**: 어둠속성 카드 치명타율 +20%, 회피율 +10%

### 해신 포세이돈
- **신기**: 신기 포세이돈
- **해금 확률**: 1%
- **아티팩트 모드 처리**: 아이스브레이크 대체
- **효과**: 기절 대상 대미지 2.5배

### 투신 아레스
- **신기**: 신기 아레스
- **해금 확률**: 1%
- **아티팩트 모드 처리**: 더블어택 대체
- **효과**: 일반 공격 대미지 2.5배

### 창조신 아스테아
- **신기**: 신기 아스테아
- **해금 확률**: 0.1%
- **아티팩트 모드 처리**: 홀리플레임버스트 대체
- **효과**: 작열/디바인 전소모 스킬 추가위력 3배

## 4. 엔드리스 보스 등장 규칙

- 엔드리스 기본 순환 보스는 인조 마신 → 사랑의 여신 아이리스 → 저주의 여신 아이리스 → 고대신 파라오 → 마신 벨제뷔트 → 창조신 아스테아 순서다.
- 37스테이지 이후에는 아래 숨은 보스가 30% 확률로 끼어든다.
  - 인조 마신 대신 꽃의 여신 플로라
  - 사랑의 여신 아이리스 대신 사신 그레이
  - 저주의 여신 아이리스 대신 뇌신 토르
  - 고대신 파라오 대신 해신 포세이돈
  - 마신 벨제뷔트 대신 투신 아레스
- 숨은 보스를 잡으면 대응 보너스 초월 카드가 10% 확률로 카오스 룰렛 풀에 추가된다.

## 6. AI 패턴 요약

- 인조 마신: 10턴째 파괴의형태를 확정 사용한다.
- 사랑의 여신 아이리스: 7턴째 소울드레인, 평소 홀리레이와 더홀리를 섞는다.
- 저주의 여신 아이리스: 10턴째 아포칼립스.
- 고대신 파라오: 5턴마다 고대의저주.
- 마신 벨제뷔트: 7턴과 14턴에 제노사이드.
- 창조신 아스테아: 1턴 일반 공격, 2턴 저지먼트, 이후 홀리레이와 차지형 디바인블레이드를 섞는다.
- 꽃의 여신 플로라: 5턴과 10턴에 제네시스블룸.
- 사신 그레이: 14턴 디멘션제로, 4의 배수 턴에는 영혼절단/차원절단 랜덤.
- 뇌신 토르: 10턴 썬더러쉬.
- 해신 포세이돈: 5턴 어비스블레싱, 10턴 어비스프레셔, 15턴 디바우러.
- 투신 아레스: 3턴과 8턴에 테라소드 또는 마그마이럽션 차지를 시작하고 다음 턴에 발사한다.
