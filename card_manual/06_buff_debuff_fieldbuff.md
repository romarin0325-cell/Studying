# 6편 버프·상태이상·필드버프

이 문서는 전투 중 실제로 쓰이는 버프와 상태이상을 이름 기준으로 정리한 참고표입니다.

## 1. 필드버프 공통 규칙

- 기본 동시 유지 수는 3개다.
- 버프오버로드가 있으면 동시 유지 수가 5개가 된다.
- 이미 있는 필드버프를 다시 걸면:
  - 지속 턴이 없는 버프는 "이미 존재" 처리만 하고 유지한다.
  - 지속 턴이 있는 버프는 남은 턴을 갱신한다.
- 상한을 넘기면 가장 오래된 필드버프부터 사라진다.
- 필드버프는 플레이어 카드에게만 적용된다.
- 팬텀은 필드버프를 전혀 받지 않는다.

## 2. 필드버프 수치표

| 필드버프 | 기본 효과 |
|---|---|
| 태양의축복 | ATK +30%, MATK +30%, 치명타 배율 +0.6 |
| 달의축복 | MATK +30%, 회피율 +15% |
| 성역 | MATK +30%, MDEF +30% |
| 여신강림 | ATK +30%, MATK +30%, DEF +30%, MDEF +30% |
| 운명의서약 | ATK +30%, MATK +30%, DEF +30%, MDEF +30% |
| 대지의축복 | ATK +25%, MATK +25% |
| 트윙클파티 | ATK +20%, 치명타율 +15% |
| 스타파우더 | DEF +40%, MDEF +40% |
| 발렌타인 | DEF +50%, MDEF +50% |
| 아레나 | 일반 공격 대미지 2배 |
| 사신강림 | 치명타율 +40%, 치명타 배율 +0.4 |
| 질풍 | 치명타율 +20%, 회피율 +20% |

## 3. 1턴 버프

이 버프들은 카드 자신의 턴이 시작될 때 지속 턴이 줄어든다.

| 버프 | 효과 |
|---|---|
| 회피태세 | 회피율 +50% |
| 배리어 | 물리 공격 1회 무효 |
| 매직가드 | 마법 공격 1회 무효 |
| 가드 | 받는 대미지 50% 감소 |

## 4. 일반 상태이상

| 상태이상 | 효과 |
|---|---|
| 약화 | ATK -20% |
| 침묵 | MATK -20% |
| 저주 | MDEF -20% |
| 유혹 | MDEF -20% |
| 암흑 | DEF -20% |
| 부식 | DEF -20% |
| 기절 | 다음 행동 불가 |

추가 규칙:

- 암흑과 부식이 함께 있으면 DEF는 총 40% 감소한다.
- 저주와 유혹은 함께 적용되면 MDEF가 총 40% 감소한다.
- 어쌔신네일이 있으면 암흑/부식 감소량이 2배가 되어 DEF 감소가 최대 80%까지 커진다.
- 섀도우볼이 있으면 암흑이 MDEF도 깎는다.
- 커스 모드에서는 위 감소량 자체가 2배가 된다.

## 5. 스택형 상태

| 상태 | 기본 상한 | 강화 시 상한 | 강화 효과 |
|---|---:|---:|---|
| 작열 | 3 | 5 | 오버플레임이 있으면 부여량도 2스택 |
| 디바인 | 3 | 5 | 오버디바인이 있으면 부여량도 2스택 |

작열과 디바인은 다음 같은 방식으로 소비된다.

- 1스택 고정 소모
- 전부 소모 후 스택 수만큼 추가 배율
- 소모 후 다른 상태이상으로 변환
- 소모 후 필드버프 발동

## 6. 신기와 상태 상호작용

| 신기 | 실제 효과 |
|---|---|
| 신기 아이리스 | 디바인 스택당 현재 공격 타입 기준 DEF 또는 MDEF를 10%씩 추가 관통 |
| 마신기 아이리스 | 작열 스택당 현재 공격 타입 기준 DEF 또는 MDEF를 10%씩 추가 관통 |
| 신기 토르 | 스타파우더 수치 2.5배 |
| 신기 플로라 | 대지의축복 수치 2.5배 |
| 신기 그레이 | 어둠속성 카드 치명타율 +20%, 회피율 +10% |
| 신기 포세이돈 | 기절 대상 대미지 2.5배 |
| 신기 아레스 | 일반 공격 대미지 2.5배 |
| 신기 아스테아 | 작열/디바인 전소모 스킬의 추가 위력 3배 |
| 신기 파라오 | DEF +50%, MDEF +50%, 회피율 -50%(최저 0%) |
| 마신기 벨제뷔트 | 전설/초월 카드 사망 시 물리 4배율 자폭 |

## 7. 실전 해석 팁

- 필드버프가 같은 스탯에 여러 개 걸리면 해당 스탯 배율에 더해서 계산한다.
- 버프/디버프가 적혀 있는 공격 스킬은 대개 대미지 계산 후 상태를 부여한다.
- 작열/디바인 "소모"는 대미지 계산 때 먼저 예약되고, 실제 상태 제거는 피해 직후에 적용된다.
- 회피태세, 배리어, 매직가드, 가드는 모두 1턴성이라 턴을 넘기면 빠르게 사라진다.
