# 10편 Lumi AI·학습 시스템·최근 변경점

## 1. 시스템 분리

현재 AI 관련 기능은 4갈래다.

1. 루미 개인과외
2. 루미 질문하기(일반 채팅)
3. TOEIC 해설 후 질문하기
4. 루미 데이트

이들 모두 `card/api.js`를 공유하지만, 프롬프트와 모델/검색 설정은 다르다.

## 2. 모델 체계

현재 선택 가능한 모델은 3개다.

- **Pro**: 검색 가능, thinking budget 사용
- **Flash**: 검색 가능, flashLike
- **Flash Lite**: 검색 불가, flashLike


### 공통 규칙

- 일반 질문은 검색 ON/OFF 토글을 가진다
- TOEIC 질문은 세션 정보 우선 답변, 필요 시에만 검색 보조
- Flash 계열은 재시도 2회, Pro는 1회
- 요청 타임아웃은 120초

## 3. 개인과외 프롬프트

`getTutoringContent()`는 아래 구조로 프롬프트를 만든다.

1. `LUMI_PERSONA`
2. 오해 유발 이벤트 지시문(30%, 30스테이지 이상 50%)
3. `LECTURE_FORMAT`
4. 대상 단어 또는 숙어 정보

### 현재 개인과외 강의 형식

- 도입
- 본문
- 암기 비법
- 용법과 뉘앙스
- TOEIC 스타일 예문
- 심쿵 포인트
- 마무리 멘트

## 5. 요약

현재 버전의 루미 인공지능은 다음과 같이 분류되어 작동합니다.

- **일반 질문**: 게임 내에서 궁금한 점을 답변해 주며 필요시 외부 지식을 검색할 수 있습니다.
- **개인 과외**: 단어를 쉽게 외울 수 있도록 재미있는 연상법과 함께 강의를 진행합니다. 가끔 오해를 부르는 귀여운 실수를 할 수도 있습니다.
- **TOEIC 질문**: 방금 푼 실전마법연습의 문항과 해설을 기반으로 정확한 답변을 제공합니다.
- **데이트**: 특별한 연출과 대화를 통해 루미와 교감할 수 있는 모드입니다.
