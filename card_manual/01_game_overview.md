# 1편 게임개요

## 기준 버전

- 코드 기준: `card/`
- 기준 커밋: `62354ed` (`origin/main`, PR #329 포함)
- 보정 대상 구문서: 다운로드된 4개 밸런스 문서

## 한 줄 요약

이 게임은 3장 덱으로 진행하는 1:1 카드 RPG이며, 티켓으로 카드를 뽑고 덱을 편성한 뒤, 전투와 퀴즈·미션·보너스 카드 해금이 서로 연결되는 구조다. 전투는 필드버프, 상태이상 스택, 특성 시너지, 초월 카드, 아티팩트 훅이 합쳐져 밸런스가 결정된다.

## 시스템 구성

| 레이어 | 담당 파일 | 역할 |
|---|---|---|
| 데이터 | `card/data.js` | 카드, 적, 버프 이름, 초월/보너스 데이터 |
| 전투 계산 | `card/logic.js` | 스탯 계산, 피해 공식, 특성/이펙트 의미, 적 AI |
| 전투 런타임 | `card/battle_runtime.js` | 턴 진행, 스킬 실행, 필드버프 적용, 사망 처리 |
| 메타 진행 | `card/rpg_features.js` | 새 게임, 미션, 보너스 풀, 카오스/드래프트, 해금 |
| UI | `card/index.html` | 메뉴, 전투 화면, 가챠, 미션, Lumi 모달 |
| AI 호출 | `card/api.js` | 개인과외, 질문하기, TOEIC 질문, 데이트, Gemini 모델 제어 |

## 기본 플레이 루프

1. 모드와 게임 타입(`challenge` 또는 `endless`)을 고른다.
2. 티켓으로 카드를 뽑고 3장 덱을 만든다.
3. 전투에서 승리해 티켓, 퀴즈 보상, 초월 후보, 이벤트 카드를 얻는다.
4. 보너스 카드 풀, 혼돈의 축복, 대현자의 축복, 아티팩트를 이용해 런을 강화한다.
5. 월간/주간 미션과 실전마법연습, 숨은 보스, 꿈의회랑으로 메타 해금을 늘린다.

## 구현상 먼저 알아야 할 현재 규칙

- MP 자동 회복 상수는 정의만 있고 전투 루프에서는 호출되지 않는다.
- 적 공격은 회피, 배리어, 매직가드, 가드 판정을 거친다.
- 플레이어 공격은 현재 구현상 적의 회피/가드/배리어/매직가드 체크를 거치지 않고 직접 대미지를 계산한다.
- 필드버프는 기본 3개, `buff_overload` 보유 시 5개까지 유지된다.
- `burn`/`divine`만 스택형 상태이고, 나머지 버프/디버프는 존재 플래그 또는 1턴 지속값으로 처리된다.
