## 4. 장비 설계

- 보스 승리 후 상점 1회 등장
- 슬롯 구성:
  - `슬롯 1`: 무기 고정
  - `슬롯 2`: 방어구 고정
  - `슬롯 3`: 무기 70% / 방어구 30%
- 리셋:
  - 상점당 2회
  - 영어 퀴즈 필요
  - 정답 시 3개 전체 재롤
  - 오답 시 1개 부분 재롤

### 4-4. 장비 밸런스 룰

- 장비는 `플랫 스탯` 중심으로 지급한다.
- `DEF/MDEF 퍼센트 장비`는 넣지 않는다.
- 이유:
  - 레벨업 방어 성장 0과 구조 일치
  - 방어 폭주 방지
  - 장비 바꿀 때 체감이 즉시 보임

## 8. 최종 추천 콘텐츠 운영 규칙

- 출시 시점 기준 `보스 AI 로직은 기존 구현을 그대로 재사용`
- 출시 시점 기준 `축복 시스템은 메인 루프에서 제거`
